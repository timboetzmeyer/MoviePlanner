MoviePlanner ist eine Software zur Planung und Begleitung von Filmprojekten.
Die Vorbereitung von Drehterminen, die Ausarbeitung von Szenen und das Management am Drehort stehen dabei im Vordergrund.
Die Software kann als Desktop-Variante für Einzelbenutzer sowie auch als Browseranwendung für Filmteams herutergeladen werden.
